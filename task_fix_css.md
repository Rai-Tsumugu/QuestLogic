# Task: index.cssの修正と「Glassmorphism: Unknown word」エラーの解消

## 概要
`index.css` にて発生している `"Glassmorphism": Unknown word` というエラーを修正します。このエラーは、ファイルの構文が乱れているか、見えない制御文字が含まれていることが原因でCSSパーサーが正常に機能していないために発生しています。また、未定義の変数の参照などの軽微な不備も併せて修正し、デザインシステムを整えます。

## 修正内容
1.  **ファイルのクリーンアップ**: ファイル内の重複や構文エラー、不正な改行などを取り除くために、`index.css` を正しい構文で書き直します。
2.  **変数名の修正**: `--color-text-main` が `--color-text` に変更されていたため、参照箇所を修正します。
3.  **スタイルの最適化**: ガラスモーフィズム効果やボタンのアニメーションを、クリーンなCSSで再実装します。

## 完了条件
*   `index.css` の構文エラーが解消されること。
*   ブラウザでデザインが正しく表示されること。
*   「Glassmorphism」に関するエラーメッセージが表示されなくなること。
