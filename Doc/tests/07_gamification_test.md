# 07_gamification_test.md

## テスト概要
エネルギー計算ロジック、ボーナス加算、IoT解錠（モック）のテスト。

## テストケース

### 1. エネルギー計算 (Energy Calculation)
- [ ] **TC-07-001**: 基本計算 (高スコア)
    - **条件**: 時間30分, スコア90点(係数1.0), ボーナスなし。
    - **期待値**: 獲得エネルギー = 30分。
- [ ] **TC-07-002**: 基本計算 (低スコア)
    - **条件**: 時間30分, スコア20点(係数0.2)。
    - **期待値**: 獲得エネルギー = 6分。
- [ ] **TC-07-003**: ボーナス加算
    - **条件**: 時間30分, スコア90点, 解き直しボーナス(+5分)。
    - **期待値**: 獲得エネルギー = 35分。

### 2. IoT / 解錠 (Unlock Flow)
- [ ] **TC-07-004**: 解錠アクション
    - **手順**: 「遊ぶ」ボタン押下（エネルギー残高あり）。
    - **期待値**: アプリ上で解錠アニメーションが表示され、タイマーがカウントダウンを開始すること。エネルギーが減算されること。
- [ ] **TC-07-005**: エネルギー不足
    - **手順**: 「遊ぶ」ボタン押下（エネルギー残高0）。
    - **期待値**: 解錠できず、「クエストをしてエネルギーを貯めよう！」等のメッセージが表示されること。
