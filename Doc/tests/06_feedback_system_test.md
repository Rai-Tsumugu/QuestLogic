# 06_feedback_system_test.md

## テスト概要
リザルト画面での演出、AIコメント表示、親からの承認通知のテスト。

## テストケース

### 1. リザルト画面 (Result View)
- [ ] **TC-06-001**: スコア別演出
    - **条件**: AI Score 85点。
    - **期待値**: 紙吹雪などの「大成功」演出が表示されること。
    - **条件**: AI Score 30点。
    - **期待値**: 励ましの演出（「次は頑張ろう」）が表示されること。
- [ ] **TC-06-002**: コメント表示
    - **手順**: リザルト画面表示。
    - **期待値**: AIが生成した `feedback_child` のテキストが、キャラクターの吹き出しとして表示されること。

### 2. 親承認通知 (Parent Notification)
- [ ] **TC-06-003**: スタンプ通知受信
    - **手順**: 親がダッシュボードで「花丸スタンプ」を送信。
    - **期待値**: 子端末に「お母さんから花丸が届きました！」等のプッシュ通知、またはアプリ内通知が届くこと。
