# 02_camera_capture_test.md

## テスト概要
学習前後の撮影フロー、ゴースト表示、集中モードの挙動に関するテストケース。

## テストケース

### 1. Before撮影 (Quest Start)
- [ ] **TC-02-001**: グリッド表示と撮影
    - **手順**: クエスト開始ボタン押下 -> カメラ起動。
    - **期待値**: グリッド線が表示され、撮影ができること。撮影後、確認画面が表示されること。
- [ ] **TC-02-002**: 水準器チェック (Optional)
    - **手順**: スマホを傾けた状態でシャッターボタンを押す。
    - **期待値**: (仕様によるが) 警告が出るか、あるいはメタデータに傾き情報が記録されること。

### 2. 学習中 (Concentration Mode)
- [ ] **TC-02-003**: バックグラウンド移行検知
    - **手順**: タイマー作動中にアプリを閉じる(ホームに戻る) -> 1分後にアプリに戻る。
    - **期待値**: 「集中モードが中断されました」等の警告が表示され、経過時間から除外される(または記録される)こと。

### 3. After撮影 (Quest Finish)
- [ ] **TC-02-004**: Ghost Overlay表示
    - **手順**: 学習終了ボタン押下 -> カメラ起動。
    - **期待値**: Before画像が半透明でオーバーレイ表示されていること。
- [ ] **TC-02-005**: 画像送信とメタデータ
    - **手順**: After撮影を行い、送信。
    - **期待値**: 画像がアップロードされ、`startedAt`, `finishedAt` 等の正しいメタデータが送信されること。
